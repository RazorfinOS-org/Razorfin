It is **always** better to install packages with Distrobox rather than layer them with rpm-ostree. `ujust distrobox` makes it easy!~[More info](https://distrobox.it/)
Packages installed in Distrobox can be exported to appear like any other application~[View documentation](https://distrobox.it/usage/distrobox-export/)
*This isn't a distro*, this is a custom image built on  Fedora Atomic Desktop technology~[View our mission](https://ublue.it/mission/)
**Razorfin** combines the gaming optimizations of Bazzite with the modern COSMIC desktop~[View on GitHub](https://github.com/RazorfinOS-org/Razorfin)
**COSMIC** is a new desktop environment from System76, built with Rust~[Learn more](https://system76.com/cosmic)
**H.264 hardware acceleration is supported out of the box.** No tweaks necessary!
Run `fastfetch` to see detailed system information with Razorfin branding
Use `ujust --choose` to explore all available system commands and utilities
